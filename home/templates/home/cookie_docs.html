About Cookie VA

Cookie the Virtual Assistant is a web application built with python and django web development framework for the purpose of making various search engines interactive and better. Cookie just acts as a platform between various other softwares and helps you to fetch your required data in a matter of time.


# Note that Cookie is not a copyrighted web application and its developers hold no intent of stealing external data or misusing search engines.

What Cookie 1.0 Does?

Cookie is built to handle both voice searches and written keywords. This enables high efficiency and cookie can fetch data in much faster time. Cookie can search through many platforms such as Google, Wikipedia, YouTube and Google Maps. We are also about to add new amazing features to cookie in the upcoming versions. To learn more about the upcoming versions read “The New Updates” article.
